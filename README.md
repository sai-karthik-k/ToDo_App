# ToDo List App

A cross-platform **ToDo mobile application** built with **Flutter** and **Dart**.  
Keep track of your daily tasks, organize your workflow, and stay productive with a clean, intuitive interface.

---

## Features

- ✅ Add, edit, and delete tasks  
- ✅ Mark tasks as completed with a single tap  
- ✅ Search and filter tasks in real-time  
- ✅ Clean and responsive UI built with Material Design  
- ✅ Modular and reusable widget architecture for maintainability

## Getting Started

Follow these instructions to set up and run the project locally:

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install)  
- [Android Studio / VS Code](https://flutter.dev/docs/get-started/editor)  
- A connected mobile device or emulator  

### Installation

1. Clone the repository:

```bash
git clone https://github.com/sai-karthik-k/ToDo_App.git
Navigate to the project directory:

cd ToDo_App


Get the Flutter dependencies:

flutter pub get


Run the app:

flutter run
```
### Project Structure
lib/
├── home.dart       # Main UI screen for the ToDo list
├── main.dart       # Entry point of the app
├── todo.dart       # ToDo model class
└── todo_item.dart  # Reusable widget for displaying a ToDo item
